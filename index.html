    The DiscoCast Visualizer Guide

    Everything you can do with the app

    Welcome to The DiscoCast Visualizer

    A modern browser-based music visualizer powered by the Winamp MilkDrop engine (Butterchurn / WebGL). Turn any audio source into a full-screen reactive light show.

    • 1,144 bundled presets — from subtle ambient washes to peak-hour chaos.
    • Live performance suite — Auto-Gain, Kick-Lock, Energy slider, and instant "Hype Keys" for DJs and VJs.
    • Build your own visuals — the Preset Studio lets you design reactive presets from scratch, no code.
    • Sequence a live show — the Timeline Editor lets you arrange presets across time and across multiple screen zones, then play them back as a scripted performance.

    Pick a section on the left to dive deeper, or press a button on the start screen to jump in.

    Audio Sources

    Three ways to feed the visualizer:

    Live Audio

    Uses your microphone, line-in, or any connected USB device (DJ controllers, audio interfaces). When you pick Live Audio, a device selector appears so you can choose the exact input.

    Requires localhost or HTTPS and microphone permission in your browser.

    Audio File

    Load a local MP3, WAV, or FLAC. You get a transport bar with play/pause, seek, and volume. File audio is split two ways: one path to your speakers, one to the visualizer (with a 5× sensitivity boost to make visuals pop).

    Preset Studio

    Skip the visualizer entirely and go straight into the creative editor to design your own presets from scratch. See the Preset Studio section for details.

    Playing in a Club or Loud Venue

    Laptop mics in loud rooms are tough — here's what actually works, in order of how well it works:

    1. Booth Out / Record Out → USB audio interface — plug directly into the DJ mixer's record output. Zero room noise, perfect signal. Even a $35 iRig works.
    2. Loopback / BlackHole — if the DJ is playing from the same Mac, route internal audio directly to DiscoCast. 100% clean, ignores the room entirely.
    3. USB audio interface with XLR mic — better preamp, physically further from the speaker stack than a laptop mic.
    4. Built-in laptop mic — last resort. Enable Noise Gate + Kick Lock in the Tuning panel. Position the laptop away from the speaker stack — even 6 feet makes a big difference. Angle the screen away from the speakers. Setting it on something soft (a bag, a hoodie) reduces sub vibration coupling through the table. If the VU meter shows CLIPPING, no software setting can fix it — the mic is physically overwhelmed and needs to move further away.

    💡 Covering the mic to muffle sound reduces everything equally — noise and signal drop together, so AGC just cranks back up. It only helps if the meter is showing CLIPPING, where it brings the signal back under the clip ceiling.

    Browsing the Preset Library

    Open the drawer with the list icon in the bottom bar or press P. Everything about the library lives here.

    Search & Tabs

    Type in the search box to filter by name. Three tabs across the top:

    • All Presets — the full 1,144-preset library.
    • Favorites — just the ones you've hearted.
    • My Presets — anything you built in the Preset Studio.
    Favorites & Hiding

    Hover a row in the drawer to reveal two icons: a ♥ heart (favorite) and a ⊘ eye-slash (hide). On the currently-playing preset, press S to favorite or X to hide.

    Hidden presets disappear from the All tab, random picks, and auto-cycle. Toggle Show hidden at the top of the drawer to bring them back — individually, or via Unhide all.

    Navigating

    ← / → step through the full library. Arrow keys always navigate the complete library, even when favorites-only cycling is on — so you can always explore.

    Auto-Cycling

    Let the app rotate through presets automatically. Click the cycle icon in the control bar (or press C) to open the Cycling popover.

    • Auto-cycle — on/off. Default interval is 30s.
    • Random order — shuffle vs. sequential.
    • Favorites only — restrict the cycle pool to your hearted presets. Perfect for live sets — curate once, cycle through the night.
    • Blend time — how long transitions take.

    If Favorites only is on but you haven't saved any, the cycle silently falls back to the full library so you're never stuck on a blank screen.

    Live Performance Tools

    Designed for DJs and VJs — quick access to everything you need mid-set.

    Audio Tuning (press T)
    • Energy slider — master visual sensitivity, 0.2× to 5×.
    • Auto-Gain (AGC) — dynamic normalization so visuals stay "hype" regardless of input volume. A toggles it. On by default.
    • Kick-Lock — low-pass filter at 150 Hz locks visuals to the kick drum only. K toggles it.
    • Noise Gate — cuts signal processing when the room is quieter than a set threshold. Prevents visuals jittering on constant ambient noise (air conditioning, crowd hum, a loud room). Off by default. When enabled, a threshold slider appears — start low and raise it until the jitter stops. The VU meter shows GATED when it's actively blocking noise, and CLIPPING (red flash) when the mic signal is too loud for software to fix.
    • Hold Shift — momentary 2× boost for drops. Silent — doesn't reveal the UI.
    Hype Keys

    Instant visual accents, beat-synced by hand:

    • V — Strobe (white flash)
    • B — Blackout (cut to black)
    • I — Invert colors
    • H — Hide UI instantly (for clean projector feeds)
    Auto-hiding control bar

    The bottom bar fades after 3 seconds of inactivity but stays visible while you hover or while a popover is open. Move the mouse to bring it back.

    Output Settings

    Control exactly how and where the visualizer renders. Open the Output panel with the monitor icon in the control bar, or press O. Settings are saved and restored automatically on every load.

    Default settings (out of the box)
    • Resolution — Match window. Canvas always fills whatever size the browser window or app is.
    • Aspect Ratio — Free. No constraint; proportions follow the window.
    • Fill Mode — Letterbox. If a resolution is locked, black bars appear rather than stretching or cropping.
    • Virtual Camera — Off. No stream is started until you turn it on.

    Nothing to configure for normal use — defaults give you a clean full-window visualizer. Only open Output Settings when you need a specific projector resolution or want to feed the canvas into OBS.

    Render Resolution

    Pin the canvas to a fixed pixel size regardless of the browser window. Choose from the dropdown or enter a custom width × height and click Apply.

    • Match window (default) — canvas always fills the window at its natural size.
    • 1280×720 — HD. Good for older projectors or lower-spec machines.
    • 1920×1080 — Full HD. Most common projector native resolution.
    • 2560×1440 — QHD. High-end displays and LED walls.
    • 3840×2160 — 4K. GPU-intensive; test performance before a live show.
    • Custom… — type any width and height and press Apply. Useful for non-standard LED panel dimensions or portrait columns.

    When a resolution is locked, the canvas renders at that size regardless of how large or small the window is. The Fill Mode setting below controls how it fits inside the window.

    Aspect Ratio

    Constrain the canvas proportions without fixing the pixel count. Useful when the window size changes but you always want a specific shape.

    • Free — no constraint; canvas matches the window exactly.
    • 16:9 — standard widescreen. Most projectors and TVs.
    • 4:3 — classic square-ish format. Older projectors.
    • 21:9 — ultra-wide / cinema. LED banners and wide screens.
    • 1:1 — perfect square. Club screens, Instagram-format content.
    • 9:16 — portrait. Vertical LED columns, phone-format screens.
    Fill Mode

    How the locked canvas is positioned inside the window when its ratio doesn’t match:

    • Letterbox (default) — canvas is centered with black bars filling the gaps. Nothing is cropped or distorted. Best for projectors where accurate proportions matter.
    • Stretch — canvas fills the window completely, ignoring proportions. Use when the output device matches the render resolution exactly.
    • Crop — canvas fills the window and any overflow is hidden. No black bars, no distortion, but the edges are cut.
    Virtual Camera

    Toggle the Virtual Camera switch to stream the visualizer canvas as a live video source. Once active, open OBS, Zoom, Teams, or any video software and select the browser window or tab as a camera input — the visualizer appears as a live feed.

    • Status hint — turns LIVE — select this window in OBS / Zoom while active; the monitor button dot turns green.
    • OBS Browser Source — an alternative: point OBS at the app URL as a Browser Source. No toggle needed — OBS captures the page directly at any resolution.
    • macOS app note — the stream is available in the WKWebView, but routing it to a system-level virtual camera requires the OBS Virtual Camera driver to be installed separately.

    The virtual camera stream runs at 60 fps and matches the current render resolution. Stopping it releases all tracks automatically.

    Projector workflow tip

    For a live show: set Resolution to 1920×1080, Aspect to 16:9, Fill Mode to Letterbox, then press H to hide the UI and F for fullscreen. The canvas renders at the exact projector native resolution regardless of what your laptop screen does.

    Preset Studio

    The Studio is its own separate page — not part of the main visualizer. Click it from the start screen and it opens with a live canvas on the left and a tabbed control panel on the right. Everything is live — drag a slider and the visual updates instantly.

    Heads up: not every slider does something on every preset.

    MilkDrop presets are little self-contained programs. Each one has its own internal logic that can take over zoom, rotation, color — basically anything — on every frame. So when you move a slider and nothing happens, that's not broken. That preset just isn't using that parameter.

    The sliders are really starting values. Some presets read them and run with them. Others override them completely with their own code. You find out which is which by trying. That's the whole game — grab a preset you like, start turning knobs, and see what responds.

    Tip: don't start from scratch. Find a preset in the browser that's already close to what you want, load it in the Studio, and remix it from there.

    The four tabs — what each one does
    • Palette — pick a starting look: solid color, pulse/breath, color shift, slow drift, and more. 12 one-click palettes, plus three independent color swatches (Wave / Glow / Accent). Color is the one thing that works on basically every preset.
    • Motion — zoom, rotation, warp, echo, and beat sensitivity. These set the motion parameters for the preset. Some presets respond directly; others have equations that take over and you'll see little or no effect. Try it and see.
    • Wave — controls the global audio waveform that reacts to your music. Every preset has one, but some presets hide it by default (opacity at zero). If you can't see a waveform, try bumping the opacity up.
    • Images — add up to 5 image or GIF layers on top of whatever preset is running. These always work, on every preset, no exceptions. Full animation, effects, and audio reactivity per layer.
    Image Layers

    Add up to 5 image or GIF layers in a collapsible smart-accordion stack. Drag any card to reorder. Each layer has:

    • Solo / Mute — audition one layer in isolation or hide it without deleting.
    • Animation — Tile, Spin, Angle, Skew X/Y, Orbit (Circle or Lissajous path), Tunnel zoom, Bounce (beat-lift), Sway, Wander, Pan, Chromatic Aberration, Tile Width & Height, and a draggable XY anchor point.
    • Beat Shake — omnidirectional random UV jolt on every beat. Pairs with Bounce for chaotic, energetic motion.
    • Beat Fade / Pulse / Strobe — audio drives opacity (smooth fade or hard strobe cut) and size on every beat.
    • Mirror — Off, Horizontal, Vertical, Quad, or Kaleido (6-slice polar fold). Apply per tile or across the whole field.
    • Canvas Mirror — folds the entire rendered scene horizontally, vertically, or both.
    • Tint & Hue Spin — colorize any layer and animate it through a spinning hue cycle.
    • Audio Reactivity — per layer, choose which audio band drives reactive controls: Bass, Mid, Treble, or Volume. Shape the response with Linear, Squared, Cubed, or Gate curves.
    • HD uploads — toggle before uploading to resize at 2048 px (default is 1024 px).
    Animated GIFs

    Drop a .gif file into any image-layer slot and it animates inside the visualizer — not as an overlay, but as a real texture, so every layer effect (tile, mirror, spin, orbit, beat-pulse, hue spin, blend mode, etc.) applies to the GIF on every frame.

    • Speed slider — top of every GIF layer card. Range 0.25× to 4×; default 1.2× (most GIFs feel a touch slow at native speed). Drag live, no reload.
    • Native frame timing — each GIF's per-frame delays are honoured at 1×, then divided by the speed multiplier. Long-delay GIFs (10 fps memes) and short-delay ones (smooth animation) both behave naturally.
    • Stack with effects — try a GIF with Tile + Mirror Kaleido + Hue Spin for psychedelic results, or pair Beat Fade with a low speed for a strobe-feel.
    • Saved with the preset — speed is persisted, so loading a saved preset restores the exact playback rate you set.
    • File size — GIFs are stored uncompressed once decoded; very large or long GIFs can use significant memory. Trim or downsize beforehand if you hit limits.
    Chromatic Aberration

    RGB channel split with animated offset — creates a glitchy color-fringe effect. Found in the Visual Effects section (between Tint and Audio Reactivity) on each image layer.

    • Chromatic slider (0–1) — amount of red/blue offset. Uses a squared curve for responsive control at low values; you’ll see the effect within the first quarter of the slider.
    • Speed slider (0–4) — animation rate of the offset. Appears automatically when Chromatic is greater than 0. Red and blue channels animate in opposite directions while green stays anchored.
    • Works in all modes — tiled, non-tiled, and tunnel. The effect respects image bounds in non-tiled mode (no bleeding outside the image).
    • Saved with the preset — both amount and speed persist in the preset JSON.
    Pan — Left/Right & Up/Down Translation

    Each image layer has a Pan section (below Wander) for deliberate whole-group translation. When a layer is tiled, Pan always moves the entire grid as one — the pattern scrolls seamlessly at the wrap edge, never individual tiles.

    • Drift — continuous linear travel. Set X and/or Y speed (−2 to +2 UV/sec, signed so negative reverses direction). With Tile ON this creates an endless scroll — logo parade, star-field drift, ticker. With Tile OFF the image slides off-canvas.
    • Bounce — ping-pong between extremes. X and Y speeds set the oscillation rate independently; Range sets the half-amplitude. Effectively a 2D Sway with full axis control.
    • Composes freely — Pan stacks additively on top of Sway, Wander, and Orbit. Try Drift X + Orbit for a spiraling scroll, or Bounce Y + Sway for a figure-eight wobble.
    • Saved with the preset — mode, speeds, and range all persist in the preset JSON.
    Angle (Static Tilt)

    An Angle slider (directly below Spin on each layer card) sets a fixed rotation offset from −180° to +180° without any continuous spinning.

    • Pure tilt — set Spin to 0 and drag Angle to lock the image at any diagonal. Great for logos at a jaunty angle or diagonal tile grids.
    • Spin offset — when Spin is also non-zero, Angle acts as the starting angle of the rotation, so the spin begins from your chosen tilt rather than 0°.
    • Works in all modes — tiled (rotates per tile), non-tiled (rotates the whole image), and tunnel.
    • Saved with the preset — persists in the preset JSON. Presets without an Angle field default to 0° (no change).
    Skew X / Y

    Two sliders directly below Angle that apply a 2D shear transform — sliding opposite edges in opposite directions to make tiles and images parallelogram-shaped.

    • Skew X (−1 to +1) — slides the top edge left or right relative to the bottom. Positive values lean the image to the right; negative lean it to the left.
    • Skew Y (−1 to +1) — slides the right edge up or down relative to the left. Combine both axes for diamond or rhombus-shaped tile grids.
    • Pairs with Angle and Width/Height — together these four controls give you a complete 2D affine toolkit: rotate, scale, and shear any image independently.
    • Works in all modes — tiled, non-tiled, and tunnel. Applied after rotation, before sizing, in every pipeline path.
    • Saved with the preset — both values persist. Older presets default to 0 (no skew).
    Perspective Tilt

    Two sliders — Persp X and Persp Y — directly below Skew in each layer card. Unlike Skew (which keeps parallel lines parallel), Perspective makes lines converge to a vanishing point, simulating depth.

    • Persp X (−1 to +1) — left and right edges converge. Positive tilts the image away from you on the right side; negative on the left. Great for billboard lean or forced-perspective text.
    • Persp Y (−1 to +1) — top and bottom edges converge. Positive creates a classic floor-tile vanishing point receding upward; negative fans it toward you.
    • Combine with Skew + Angle — together these give you a full 2D projective toolkit. Persp Y + Tile is the go-to for infinite floor or ceiling grids.
    • Works in all modes — tiled, non-tiled, and tunnel. Applied after Skew in every pipeline path. Singularity is clamped so extreme values stay safe.
    • Zero cost at 0 — no shader code emitted unless a non-zero value is set.
    Depth Stack

    A Depth slider below the Tunnel slider (visible when Tile is ON) that offsets a layer's zoom phase in the tunnel, making it appear to fly through space at a different Z depth than other layers.

    • The classic move — two layers, same image, Tunnel speed identical, Depth = 0 on Layer 1 and Depth = 0.5 on Layer 2. The tiles zoom past each other in true parallax — no other setting achieves this.
    • Range 0–1 — 0.5 = exactly 180° phase offset (maximum parallax). Other values give subtler staggered rhythms.
    • Tunnel-only — the slider is hidden when Tile is OFF. Zero cost at 0.
    • Combine freely — each layer can have its own Depth, Spin, Tint, and Size for a full multi-plane 3D look.
    Edge / Sobel

    An Edge On/Off toggle in the Visual Effects section that replaces the image with its edge-detected outline — turning any photo into animated neon line art.

    • How it works — a 3×3 Sobel kernel samples 8 neighbouring pixels, computes the luminance gradient, and uses the magnitude as brightness. High-contrast edges glow white; flat areas go black.
    • Best combo — Edge ON + Tint color + Hue Spin. The monochrome edges take the Tint color perfectly, and Hue Spin cycles the outline through the full spectrum.
    • Layer it — one layer with Edge ON (neon outlines) blended over a second layer with the original image creates a mixed illustration style.
    • Works in all modes — tiled, non-tiled, and tunnel. Posterize stacks on top for an extra-crunchy look.
    • Zero cost when Off — no extra samples taken unless enabled.
    Beat Shake

    A Shake slider in the Audio Reactivity section (below Bounce) that jolts the image in a random direction on every beat.

    • Omnidirectional — unlike Bounce (always up), Shake fires in a new random direction each beat for a chaotic, glitchy energy.
    • Scales with audio — uses the same shaped audio signal as Pulse and Bounce, so Reactivity Source and Curve both apply.
    • Combine freely — stack with Bounce for upward-biased chaos, or with Sway for an organic tremor feel.
    • Cubic curve — the slider gives fine control at low values; big jolts live at the top end.
    Posterize

    A segmented button row in the Visual Effects section (Off / 2 / 4 / 8 / 16) that buckets each color channel into discrete steps — turning any photo into flat graphic art.

    • 2 steps — hard 2-tone silhouette. Graphic and bold.
    • 4 steps — classic pop-art look. Great with a bright Tint color.
    • 8 / 16 steps — subtle banding, still photographic but stylized.
    • Pairs with Hue Spin — the limited palette cycles through hues for an animated retro palette effect.
    • Off = zero cost — no shader code emitted when set to Off.
    Tile Width & Height

    Two sliders in each image layer card (visible only when Tile is ON, sitting below Spacing) that let you independently control the shape of each tile cell.

    • Width (0.25–4.0) — stretches or squashes tile cells horizontally. 1.0 = native image aspect ratio (the default). Push below 1.0 for narrow portrait-style cells, above 1.0 for wide panoramic ones.
    • Height (0.25–4.0) — same control on the vertical axis. Combine with Width to make square cells from a landscape image, or tall slivers from a square one.
    • Squared curve — both sliders use a squared response so the 1.0 default lands at roughly mid-travel. Fine adjustments in the lower half, big stretches at the top.
    • Composes with Spacing — changing cell shape doesn't affect gap size. Spacing still controls the gap between tiles as a fraction of each cell.
    • Saved with the preset — both values persist in the preset JSON. Presets saved before this control existed default to 1.0 and render identically.
    Workflow
    • Undo / Redo — 50-deep stack. ⌘Z / ⌘⇧Z.
    • A/B compare — hold the button to preview the saved state; release to restore.
    • Save — writes to your browser (localStorage + IndexedDB for images). Saved presets appear in the My Presets tab of the main app.
    • Reset a slider — double-click its label to snap it back to the default value instantly. Works on every slider in every layer card.
    • Preview mode — click the expand icon in the top-right corner of the topbar (or press \) to hide the panel and see the full canvas. Click anywhere on the canvas to restore the panel.
    • First-use tips — a tips modal appears when you open the editor. Dismiss it freely or click "Never show again" to stop it appearing. Reset with localStorage.removeItem('discocast_onboarding_never') in the browser console.
    • Dev HUD — press ` (backtick) to overlay frame time, estimated VRAM, active layer count, and last shader rebuild time.

    Timeline Editor

    A dedicated page for scripting a full live show. Arrange presets into timed blocks on a horizontal strip, then hit Play — the visualizer runs your sequence automatically. Launch it from the start screen or press L from inside the main app.

    Timelines & Blocks

    A timeline is a playlist of preset blocks, each with a fixed start time, duration, and blend-in transition. Blocks sit on a scrollable strip — drag them to any position, resize their right edge to change duration, or drop them into any gap. Multiple timelines can be saved and switched between.

    Zones — Multiple Screen Regions

    Split the canvas into up to four simultaneous regions, each running its own preset independently. Six predefined layouts are available:

    • Full Screen — single zone, whole canvas.
    • Left | Right — two side-by-side halves.
    • Top / Bottom — two stacked halves.
    • 4 Quadrants — four equal tiles.
    • Center + Frame — a framing overlay blended over the main canvas.
    • Top Banner — a horizontal strip blended above the main canvas.

    Click ⊞ Zones in the transport bar to pick a layout. Each zone gets its own row in the strip. All zones share the same audio signal.

    Playback & Seeking

    Press Play to start all zones simultaneously from t=0. Click anywhere on the ruler to seek — all zones restart from that point in sync. Enable Loop to repeat the full sequence automatically. The playhead advances in real time and can be watched while the visualizer plays full-screen behind the strip.

    Adding & Editing Blocks

    Click + Add Preset in the transport, or the + button on any zone row, to open the preset picker (All / Favorites / My Presets tabs with search). Each block has three inline action buttons on hover — Edit, Duplicate, Delete — plus a right-click context menu. The Edit popover lets you set Duration (5–600 s), Blend time (0–10 s), and an optional text Label.

    Saving & Managing Timelines

    New timelines are held in memory until you click Save. Once saved, the button dims to Saved and shows a dot whenever you have unsaved changes. Use + New to start fresh, or the dropdown to switch between saved timelines (a discard prompt fires if you have unsaved edits). Copy / Paste JSON buttons let you back up or share a timeline as text.

    Controls & Auto-hide

    All overlays (topbar, transport, timeline strip) fade after 3.5 s of inactivity — the same pattern as the main app. The ≡ button in the top-right corner pins them permanently; click again to unpin. Press T to toggle the strip panel on and off without touching the mouse.

    Reactivity & Audio Controls

    Every image layer has a Reactivity panel at the bottom of its card. The two settings there — Source and Curve — are the dial behind the dial. They shape the single audio signal that drives all four reactive controls on that layer: Pulse, Bounce, Beat Fade, and Strobe.

    Step 1 — Pick a Source

    Which frequency band feeds this layer's reactivity:

    • Bass (default) — sub and kick frequencies. The most punch-per-beat; most presets start here.
    • Mid — snare, vocals, guitars. Reacts on melodic and rhythmic hits mid-song.
    • Treble — hi-hats, cymbals, air. Fast and bright; great for sparkle layers.
    • Volume — total loudness across all bands. Tracks the overall energy of the mix.
    Step 2 — Shape the Curve

    The curve transforms the raw signal before it reaches any control. A gentle signal becomes more dramatic — or more surgical:

    • Linear — signal passed as-is. Controls scale directly and evenly with audio level.
    • Squared — quiet moments are suppressed; loud peaks hit harder. The most musical curve for most layers.
    • Cubed — extreme contrast. Only the very loudest peaks drive any visible reaction.
    • Gate — binary on/off at 30%. Below the threshold: zero. Above it: full signal. Controls either stay still or snap to maximum.
    What Each Control Does With the Signal
    • Pulse — grows the layer size on every peak. The curve determines whether small beats cause any growth at all — Squared suppresses gentle hits, Cubed reserves the swell for the loudest moments only.
    • Bounce — lifts the layer upward on beat. With a Gate curve it snaps up hard on kicks and drops instantly between them. With Linear it floats up and down with the groove.
    • Beat Fade — adds opacity on beat. The layer appears or brightens with the audio, then fades back when the signal drops. Combines beautifully with low base Opacity so the layer is normally invisible and only materialises on hits.
    • Strobe — unlike the others, Strobe reads the raw audio signal directly (before the Curve is applied) and compares it to its own Threshold slider. When the audio crosses the threshold, the layer's opacity hard-cuts to black — instantly. When the beat fades, it returns. This is a true strobe: no smooth ramp, just a sharp on/off flash driven by real beats.
    Example Combos
    • Kick-lock flash — Source: Bass · Curve: Gate · Strobe: 1.0 · Threshold: 0.4 · Base Opacity: 0. The layer is invisible except on every hard kick, where it flashes at full opacity. Zero ambient light between beats.
    • Melodic drift — Source: Mid · Curve: Squared · Bounce: medium. The layer lifts on snare or vocal hits, barely moves between them. Feels tied to the melody rather than the kick.
    • Ambient breathing — Source: Volume · Curve: Linear · Beat Fade: 0.6 · Base Opacity: 0.2. The layer's opacity tracks the mix energy — bright during builds, dim during breakdowns. Feels alive without being distracting.
    • Treble sparkle — Source: Treble · Curve: Cubed · Pulse: high. The layer explodes in size on hi-hat and cymbal peaks only. Everything below that frequency is invisible to it.
    • Multi-layer stack — put two layers side by side: one on Bass/Squared driving Bounce, one on Treble/Cubed driving Pulse. They react to completely different parts of the music and never compete.

    Tip: a layer with Strobe on and Beat Fade off feels like a strobe light. Flip it — Beat Fade on and Strobe off — and it feels like a soft glow that pulses with the music. Both use the same Reactivity panel; only the control you turn up changes the personality.

    Tips & Troubleshooting

    Browser support

    DiscoCast Visualizer needs WebGL 2 and the Web Audio API. Works best in Chrome or Firefox. Safari works but some presets may render differently.

    Microphone access

    Browsers only allow mic access on localhost or HTTPS. If the mic button fails, check the address bar for a blocked-permission icon and re-allow.

    Your data stays local

    Favorites, hidden presets, and saved custom presets are all stored in your browser's localStorage (and IndexedDB for images). Nothing is uploaded — clearing site data resets everything.

    Performance

    For the smoothest visuals on older machines, use F for fullscreen, close other tabs running video or WebGL, and prefer file input over live (less processing overhead).

    Projector / VJ setup

    Press H to hide the UI completely for a clean feed, then F for fullscreen. For a locked projector resolution, press O to open Output Settings — set your projector’s native resolution (e.g. 1920×1080), pick Letterbox, and the canvas renders at that size regardless of your laptop screen. Move the mouse to bring controls back.

    Preset Studio — How it works

    The Studio is its own separate page, not part of the main visualizer. Hit E or click it from the start screen. You get a live canvas on the left and a tabbed control panel on the right. Every change is instant — no save, no reload.

    Not every slider does something on every preset — and that's normal.
    MilkDrop presets are self-contained programs. Each one has its own internal equations that can take over zoom, rotation, color, or geometry on every frame. When you move a slider and nothing happens, that preset just isn't reading that parameter. The sliders are starting values — some presets use them, others override them completely with their own code. The only way to know is to try. That's the whole point of remixing.

    Best workflow: find a preset in the browser that's already close to what you want, open it in the Studio with the pencil icon, and tweak from there. Starting from a blank slate is harder.

    The four tabs

    • Palette — pick a base look: solid color, pulse, drift, color shift, and more. 12 one-click color palettes plus three independent color swatches. Color is the one thing that works on basically every preset.
    • Motion — zoom, rotation, warp, echo, beat sensitivity. Some presets respond directly; others have equations that take over and you'll see little or no effect. Try it and see.
    • Wave — the global audio waveform that reacts to your music. Every preset has one, but some hide it by default (opacity at zero). If you can't see a waveform, bump the opacity up.
    • Images — up to 5 image or GIF layers on top of whatever preset is running. These always work, on every preset, no exceptions.

    Image Layers

    Up to 5 image or GIF layers, each compositing over the visualizer with its own blend mode and full animation controls.

    • Add Layer — accepts JPG, PNG, GIF, WebP, SVG, AVIF.
    • Drag to reorder — grab the ⠿ handle. Top of list = drawn on top.
    • Solo / Mute — audition one layer or hide it without deleting.
    • Reset ↻ — restores all animation fields to default, keeps the image. Undoable.
    • HD toggle — enable before uploading for 2048 px (default 1024 px).

    Animated GIFs

    GIFs animate as real GPU textures — every effect (mirror, tunnel, hue spin, audio reactivity) applies to every frame in real time.

    • Speed slider — 0.25× to 8×. Default is 1.2× because most web GIFs are encoded slow (10 fps native). Drag right for fast-forward, left for slow-motion freeze-frame feel.
    • All effects work — Tile, Mirror, Kaleido, Tunnel, Pulse, Strobe, Hue Spin, Chromatic, Edge — everything applies per frame.
    • GIFs are not resized on upload — they load at native resolution. A 1200×1200 GIF uses significantly more GPU memory than a 512×512 one.
    Performance guide:
    • 1–2 GIF layers — smooth on any device.
    • 3–4 GIF layers — fine on a dedicated GPU, may stutter on integrated graphics.
    • 5+ GIFs or large GIFs (1024 px+) — expect slowdown regardless of GPU. Use smaller or fewer-frame GIFs.
    • Adding audio reactivity to a GIF layer costs extra GPU work per frame. If things slow down, reduce the number of audio-reactive GIF layers first.
    GIF Optimizer (auto-prompt on upload): When you drop a GIF that is large (>256 px) or has many frames (>10), the Optimizer appears automatically before the layer is created. It lets you:
    • Keep every Nth frame — reduce frame count. Delays are scaled so animation stays at the same perceived speed.
    • Resize to 128 / 192 / 256 px — shrink large GIFs. Saves significant GPU memory.
    • Live GPU savings preview — shows before/after memory estimate so you know the trade-off before committing.
    • Click Use Original to skip optimization and load as-is.

    Transform

    • Size — scales the image or each tile. Pulse adds audio-reactive size.
    • Angle — static rotation offset, composes with Spin.
    • Skew X / Y — shears the image into a parallelogram. Works in all modes.
    • Persp X / Y — projective warp, lines converge to a vanishing point. Persp Y + Tile = infinite floor grid.
    • Width / Height — independent tile cell shape (Tile mode only).

    Motion

    • Spin — continuous rotation. Per-tile when Tile ON.
    • Orbit — circular path around anchor. Switch to Lissajous for figure-8 / clover paths.
    • Bounce — bass-driven upward displacement on every beat.
    • Beat Shake — random-direction UV impulse 24×/sec, scales with audio.
    • Sway — sinusoidal horizontal pendulum.
    • Wander — dual-sine organic drift in X and Y.
    • Pan — Drift = continuous scroll; Bounce = ping-pong around anchor.

    Tunnel & Depth Stack

    • Tunnel — seamless zoom-through animation. Tile must be ON. Negative = zoom out.
    • Depth — offsets a layer's zoom phase. 0.5 = 180° = maximum parallax between layers.
    Classic move: Two layers, same image, same Tunnel speed. Layer 1 Depth = 0, Layer 2 Depth = 0.5. Tiles zoom past each other at different Z depths.

    Visual Effects

    • Tint + Hue Spin — colorize and cycle through the full spectrum continuously.
    • Chromatic — RGB channel split with animated offset. Glitchy color fringe.
    • Posterize — Off / 2 / 4 / 8 / 16 color steps. Pop-art quantize effect.
    • Edge / Sobel — 3×3 Sobel kernel replaces image with luminance edge outline. Neon line-art. Best with Tint + Hue Spin.
    Best combo: Edge ON · Tint = vivid color · Hue Spin = 0.3 · Blend = Additive.

    Mirror

    • Off / H / V / Quad / Kaleido — fold the image symmetrically.
    • Per Tile — folds inside each tile cell.
    • Whole Image — folds the entire tiled field as one unit.

    Audio Reactivity

    • Source — Bass / Mid / Treble / Volume. Drives Pulse, Bounce, Shake, Strobe, Opacity Pulse.
    • Curve — Linear / Squared / Cubed / Gate. Shapes how the signal maps to motion.
    • Strobe — hard binary opacity cut. Threshold sets the trigger level.
    Tip: One layer on Bass, another on Treble — they react to different parts of the beat for automatic depth.

    Workflow

    • Undo / Redo — ⌘Z / ⌘⇧Z. 50-step history. Every action is undoable.
    • A/B compare — hold the A/B button to preview the last saved state.
    • Save — persists preset + images to browser storage. Appears in My Presets.
    • Focus mode — expand icon (or \) hides the panel. Click canvas to restore.

    Curated Combos

    Neon Line-Art

    Edge ON · Tint → vivid color · Hue Spin → 0.3 · Blend → Additive.
    Stack a second layer (Edge OFF) underneath at low opacity for a ghost fill behind the outlines.

    3D Parallax Tunnel

    Two layers, same image, same Tunnel speed.
    Layer 1: Depth = 0. Layer 2: Depth = 0.5 + different Tint color.
    Tiles zoom past each other at different Z depths — true parallax without any 3D math.

    Infinite Floor Grid

    Tile ON · Persp Y → 0.6 · Tunnel → 0.5 · Spin → 0.05 · Mirror → Quad.
    Add Bounce → 0.15 (Bass) so the floor heaves on every kick.

    Pop-Art Pulse

    Posterize → 4 · Tint → bright primary · Hue Spin → 0.1 · Bounce → 0.3 · Curve → Cubed.
    The Cubed curve means only hard hits trigger the jump — clean on sparse beats, explosive on drops.

    Glitch Stack

    Layer 1: Chromatic → 0.6 · Beat Shake → 0.08 · Blend → Overlay.
    Layer 2: Edge ON · Strobe → 0.8 · Blend → Additive · Source → Treble.

    Kaleidoscope Vortex

    Tile ON · Mirror → Kaleido · Spin → 0.4 · Orbit radius → 0.08 · Hue Spin → 0.2.

    Breathing Logo

    Single image, Tile OFF · Size → 0.35 · Pulse → 0.25 · Bounce → 0.1 · Source → Bass · Curve → Squared.

    Warp Speed Stars

    Star field texture · Tile ON · Tunnel → 1.8 · Spacing → 0.3 · Size → 0.15 · Blend → Additive.

    Liquid Mirror

    Tile ON · Mirror → Quad, Scope → Whole Image · Sway Amt → 0.08 · Wander Amt → 0.04. No Tunnel.

    Diamond Tile Grid

    Tile ON · Angle → 45° · Skew X → 0.3 · Skew Y → −0.3 · Width → 0.7 · Height → 0.7 · Spin → 0.03.

    Depth Hue Shift

    Three layers, same image, Tunnel = 0.6.
    Layer 1: Depth = 0 · Tint = cyan. Layer 2: Depth = 0.33 · Tint = magenta. Layer 3: Depth = 0.66 · Tint = yellow.

    Animated GIF Loop

    Drop a GIF · Speed → 1.2 · Mirror → Kaleido · Hue Spin → 0.2 · Blend → Additive.
    Or: Speed → 0.25 · Strobe ON · Source → Bass — frames strobe to the beat in slow-motion.

    Bass Strobe + Treble Shimmer

    Layer 1: Opacity Pulse → 0.6 · Source → Bass · Curve → Cubed.
    Layer 2: Hue Spin → 1.0 · Chromatic → 0.2 · Source → Treble · Blend → Screen.
