Quick Start

From launch to visuals in under a minute.

1 · Load a song

Click the folder icon in the bottom control bar, or press Space to load a file. MP3, WAV, OGG, FLAC, AAC, and M4A are supported. DiscoCast analyses the audio in real time — no pre-processing.

2 · Pick a preset

Click the preset name in the control bar to open the browser. You have 1,144 MilkDrop presets plus any you've saved. Press R at any time to jump to a random one. Use ← / → to step through.

3 · Add an image layer (optional)

Open the Preset Studio with the edit icon (or E). Click Add Layer and drop an image or GIF. The layer composites over the visualizer in real time. Close the editor to see the full canvas.

4 · Go full screen

Press F for fullscreen, or connect to a projector and drag the window to that display. Use Output Settings (O) to lock the render resolution to match the projector's native size.

Interface Overview

DiscoCast has three main areas — the visualizer canvas, the control bar, and the Preset Studio editor.

Control Bar

The thin bar at the bottom of the canvas. From left to right:

  • Load file — open an audio file.
  • Play / Pause — Space.
  • Preset name — click to open the preset browser.
  • ← → — step to previous / next preset.
  • R — random preset.
  • Edit icon — open the Preset Studio editor. E.
  • Output icon — open Output Settings. O.
  • Fullscreen — F.

Preset Studio Editor

A side panel that slides in from the right. Contains the butterchurn parameter controls (Palette, Motion, Wave, Feel) at the top, followed by the Image Layers stack. Close it with the × or press E again. Use the expand icon (or \) to enter Focus Mode — hides the panel so you can see the full canvas; click the canvas or press \ again to restore.

Preset Studio — Editor

The editor lets you customize any preset and add animated image layers. Changes are live — you see the result on every slider move.

  • Palette — base color variations, 12 one-click palettes, three color swatches (Wave / Glow / Accent).
  • Motion — Zoom, Rotation, Warp amount & speed, Echo.
  • Wave — 8 shapes, size, opacity, thickness, additive blend.
  • Feel — Decay (trail length), Gamma (brightness), warp scale/speed.
  • Images — up to 5 image or GIF layers. See below.
Tip: Use ⌘Z / ⌘⇧Z freely — every control change is undoable. The A/B button lets you compare your edits against the last saved state.

Image Layers

Add up to 5 image or GIF layers. Each layer composites over the visualizer using its own blend mode, opacity, and full set of animation controls.

Adding & managing layers

  • Add Layer — button at the top of the Images section. Accepts JPG, PNG, GIF, WebP, SVG, AVIF.
  • Drag to reorder — grab the ⠿ handle on the left edge of any card. Top of the list = drawn last = on top.
  • Collapse / expand — click the card header or chevron. Adding a new layer auto-collapses others.
  • Solo — shows only that layer (and any other soloed layers). Click again to unsolo.
  • Mute — hides the layer without removing it.
  • Reset — ↻ button restores all animation fields to default while keeping the image. Undoable.
  • Delete — 🗑 button. Asks for confirmation. Undoable.
  • HD toggle — enable before uploading for 2048 px max (default is 1024 px). HD badge appears in the card header.

Transform Controls

All transform controls apply in UV space — they affect how the image is mapped onto the screen, not the image pixels themselves. They compose together.

Size & Angle

  • Size (0–1) — scales the image or each tile. Pulse makes size react to the audio beat.
  • Angle (−180° to +180°) — static rotation offset. Composes with Spin so you can lock a tilt while the image also rotates.
  • Spin (−2 to +2 rad/s) — continuous rotation. Per-tile when Tile mode is ON.

Skew X / Y

Shears the image — slides opposite edges in opposite directions to make it parallelogram-shaped.

  • Skew X — slides the top edge left or right relative to the bottom.
  • Skew Y — slides the right edge up or down relative to the left.
  • Combine both for diamond or rhombus tile grids. Works in all modes — tiled, non-tiled, tunnel.

Perspective Tilt (Persp X / Y)

Unlike Skew (which keeps parallel lines parallel), Perspective makes lines converge to a vanishing point — simulating a surface receding into the distance.

  • Persp X (−1 to +1) — left/right edges converge. Positive leans the right side away; negative leans the left. Billboard tilt.
  • Persp Y (−1 to +1) — top/bottom edges converge. Positive = classic floor-tile vanishing point receding upward.
  • Combine Persp Y + Tile for an infinite floor grid. Combine with Angle + Skew for full projective control.
  • Singularity is clamped — extreme values are safe.
Best combo: Tile ON · Persp Y = 0.5–0.8 · Spin = slow · Tunnel ON → animated infinite floor with tiles zooming toward you.

Tile Width & Height

Independently controls the shape of each tile cell (visible only when Tile is ON).

  • Width (0.25–4.0) — stretch or squash cells horizontally. 1.0 = native image aspect.
  • Height (0.25–4.0) — same on the vertical axis.
  • Composes with Spacing — changing cell shape doesn't affect gap size.

Motion Controls

Motion controls animate the layer's position and movement over time. They all compose additively.

Orbit

  • Orbit radius — circular path around the anchor point.
  • Path: Circle — uniform circular orbit.
  • Path: Lissajous — reveals Freq X, Freq Y, and Phase sliders. The ratio between Freq X and Freq Y determines the figure shape (2:3 = figure-8, 3:4 = four-leaf clover). Phase rotates the figure.

Sway & Wander

  • Sway — sinusoidal horizontal pendulum. Amt controls how far; Speed controls rate.
  • Wander — dual-sine drift in both X and Y for an organic floating feel.

Pan

  • Off — no pan.
  • Drift — continuous linear scroll in X and / or Y. Speed is signed — negative reverses direction. Great for endless tile scroll or crawling logos.
  • Bounce — ping-pong within a range around the anchor. Range slider sets the half-amplitude.

Bounce & Beat Shake

  • Bounce — audio drives an upward Y displacement on every beat. Always moves up; use Sway for horizontal.
  • Beat Shake — fires a random-direction UV impulse 24× per second, scaled to the audio signal. Omnidirectional chaos. Pairs with Bounce for upward-biased energy.

Tunnel & Depth Stack

Tunnel mode zooms the tiled image toward (or away from) the viewer in a seamless looping animation — the defining DiscoCast look.

Tunnel

  • Speed (−2 to +2) — how fast tiles zoom. Positive = zooming in (toward viewer). Negative = zooming out. Zero = off.
  • Uses a seamless two-layer crossfade so the zoom never jumps or resets — tiles appear to fly past you forever.
  • Tunnel is only active when Tile is ON. The Tunnel slider is hidden when Tile is OFF.
Tip: Set Tunnel speed to 0.5–1.0 for a gentle pull. Push to 2.0 for warp-speed intensity. Combine with Spin for a spinning vortex.

Depth Stack (Z-phase offset)

Offsets a layer's zoom phase in the tunnel, making it appear to fly through space at a different Z depth than other layers.

  • Range 0–1. 0.5 = exactly 180° phase offset = maximum parallax between layers.
  • At 0 (default) the layer is in phase with any other Depth=0 layers — they zoom in lockstep and look flat.
  • Tunnel-only. The Depth slider is hidden when Tile is OFF.
Classic move: Two layers, same image, same Tunnel speed.
Layer 1 → Depth = 0  ·  Layer 2 → Depth = 0.5
Result: tiles appear to zoom past each other at different Z depths — genuine 3D parallax.

Visual Effects

Found in the Visual Effects section of each layer card, below Tint.

Tint & Hue Spin

  • Tint color — colorizes the layer by multiplying the sampled color. White = no tint.
  • Hue Spin (0–2 rad/s) — animates the tint through a full color cycle. Slow values pulse gently; fast values strobe through the spectrum.

Chromatic Aberration

  • Splits the RGB channels — red and blue shift in opposite directions, green stays centered. Creates a glitchy color-fringe effect.
  • Chromatic slider (0–1) — amount of split.
  • Speed sub-slider — how fast the split animates. Appears when Chromatic > 0.

Posterize

  • Quantizes each color channel to a fixed number of steps: Off / 2 / 4 / 8 / 16.
  • 2 steps = hard 2-tone silhouette. 4 steps = classic pop-art. 8–16 = subtle banding.
  • Applied after Tint — combine with Hue Spin for an animated retro palette effect.
  • Zero shader cost when set to Off.

Edge / Sobel

Replaces the image with its edge-detected outline — turning any photo into animated neon line art.

  • A 3×3 Sobel kernel samples 8 neighbouring pixels, computes the luminance gradient, and uses the magnitude as brightness. High-contrast edges glow; flat areas go black.
  • Toggle: Off (default) or On.
  • Zero extra texture samples when Off.
Best combo: Edge ON → set Tint to any vivid color → Hue Spin = 0.3. The monochrome edges take the color perfectly, and the outlines cycle through the full spectrum.
Layer trick: Layer 1 = Edge ON (neon outlines) blended over Layer 2 = original image → mixed illustration style.

Mirror

Folds the image UV symmetrically. Each layer has its own Mirror control, independent of the scene-level Canvas Mirror.

  • Off — no fold.
  • ↔ H — horizontal fold (left = right).
  • ↕ V — vertical fold (top = bottom).
  • ⊞ Quad — both H and V — four identical quadrants.
  • ✦ Kaleido — 6-slice polar fold for psychedelic kaleidoscope patterns.

Mirror Scope

Appears when any Mirror mode is active:

  • Per Tile — folds inside each individual tile cell. Good for geometric tile patterns.
  • Whole Image — folds the entire tiled field as one unit. Creates a large-scale symmetry across the full canvas.

Audio Reactivity

Each image layer has its own Reactivity panel at the bottom of its card. This controls what audio band drives reactive controls (Pulse, Bounce, Shake, Strobe, Opacity Pulse) for that layer.

Source

  • Bass (default) — low-frequency energy. Kicks and bass hits. Most impactful for beat-driven motion.
  • Mid — midrange. Snares, vocals, chord hits.
  • Treble — high frequency. Hi-hats, cymbals, sibilance.
  • Volume — overall RMS. Tracks the general loudness envelope.
Tip: Set one layer to Bass and another to Treble — they'll move to different parts of the beat, creating depth without any extra effort.

Curve

Shapes how the raw audio signal maps to motion intensity:

  • Linear — direct 1:1 mapping. Smooth response.
  • Squared — softens quiet moments, emphasises peaks. Good for punchy bass.
  • Cubed — even more peak-biased. Only the hardest hits trigger motion.
  • Threshold (Gate) — binary: below 30% = nothing, above = full. Hard-gate strobe feel.

Strobe

  • Hard binary opacity cut triggered by the raw audio signal (pre-curve).
  • Strobe slider — how much of the opacity is driven by the gate.
  • Threshold sub-slider — the signal level that triggers the cut. Appears when Strobe > 0.

Workflow

Save & Load Presets

  • Save — writes the current preset (including all image layers) to browser storage. Images are stored in IndexedDB so they survive page reloads.
  • My Presets — saved presets appear in the main app's preset browser alongside the built-in MilkDrop presets.
  • Export / Import — use the export button to download a preset as a JSON file. Share it with anyone; they can import it via the import button.

Undo / Redo

  • 50-step undo history. ⌘Z to undo, ⌘⇧Z to redo.
  • Every control change, reorder, add, delete, and reset is a single undoable step.

A/B Compare

  • Hold the A/B button to preview the last saved state of the preset.
  • Release to return to your current edits. Useful for comparing a tweak against the baseline.

Focus / Preview Mode

  • Click the expand icon in the editor topbar (or press \) to hide the editor panel — the canvas fills the full window.
  • Click anywhere on the canvas (or press \ again) to restore the editor.
  • Useful for checking how a preset looks without the panel taking up space.

Output Settings

Open with the monitor icon in the control bar, or press O. Settings are saved and restored automatically.

Render Resolution

  • Match window (default) — canvas always fills the window at its natural size.
  • 1280×720 / 1920×1080 / 2560×1440 / 3840×2160 — lock to a specific size. Good for projectors with a known native resolution.
  • Custom — type any width and height. Useful for non-standard LED panels or portrait columns.
Live show tip: Set the resolution to match your projector's native size before connecting. This prevents the canvas from blurring when the OS scales the window.

Aspect Ratio & Fill Mode

  • Free — no constraint; canvas matches the window.
  • 16:9 / 4:3 / 21:9 / 1:1 / 9:16 — constrain proportions independent of window size.
  • Letterbox — black bars fill the gaps. Nothing is cropped or distorted.
  • Stretch — fills the window completely, ignoring proportions.
  • Crop — fills the window; edges are cut. No black bars.

Virtual Camera / OBS

  • Enable the Virtual Camera toggle to make the visualizer canvas available as a camera source in OBS, Zoom, or any video app.
  • In OBS: Add a Window Capture source pointing to the DiscoCast window, or use the virtual camera if your setup supports it.
  • For best results, lock the render resolution to match your OBS canvas size before starting a stream.

Tips & Combos

Curated starting points for the most visually striking results.

Neon Line-Art

Any photo → animated glowing outlines.
Edge ON · Tint → vivid color · Hue Spin → 0.3 · Blend → Additive
Stack a second layer (same image, Edge OFF) underneath at low opacity for a ghost-fill effect.

3D Parallax Tunnel

Two layers of the same image appear to zoom past each other at different depths.
Layer 1: Tile ON · Tunnel → 0.8 · Depth → 0
Layer 2: Tile ON · Tunnel → 0.8 · Depth → 0.5 · Tint → different color

Infinite Floor Grid

Classic perspective grid receding to a vanishing point.
Tile ON · Persp Y → 0.6 · Tunnel → 0.5 · Spin → 0.05 · Mirror → Quad

Animated GIF Tricks

GIFs animate as a real texture — all effects apply per frame.
Try: GIF · Tile ON · Mirror → Kaleido · Hue Spin → 0.5 · Tunnel → 0.3
Or: GIF Speed → 0.25 · Strobe ON · Bass source → frames strobe to the beat.

Pop-Art Pulse

Bold graphic style that hammers on every kick.
Posterize → 4 · Tint → bright primary · Hue Spin → 0.1 · Bounce → 0.3 · Pulse → 0.4 · Curve → Cubed

Glitch Stack

High-energy digital chaos.
Layer 1: Chromatic → 0.6 · Beat Shake → 0.08 · Blend → Overlay
Layer 2: Edge ON · Strobe → 0.8 · Blend → Additive · Source → Treble

Performance & Limits

DiscoCast runs entirely on your GPU — there is no server rendering cost. Performance depends on your machine's WebGL capabilities.

  • 5 layers — the maximum. Runs comfortably on any modern laptop at 1080p. GPU-intensive effects (Edge/Sobel, Chromatic, Kaleido) add a small per-layer cost.
  • Image size — standard uploads are capped at 1024 px (longest side). Enable HD before uploading for 2048 px. Very large images increase VRAM usage.
  • GIFs — decoded and stored uncompressed. Very long or large GIFs can use significant memory. Trim beforehand if you hit limits.
  • Dev HUD — press ` to see live frame time (ms), VRAM estimate, layer count, and last shader rebuild time. Target: under 16ms (60 fps).
  • Mobile — works but is limited. Recommend 1–2 layers maximum on mobile devices. Use standard (1024 px) images only.
If you see frame drops: reduce layer count, disable Edge/Sobel on complex layers, lower render resolution in Output Settings, or switch large images to standard (1024 px) quality.
