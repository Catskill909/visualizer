Quick Start

1 · Load a track

Click the folder icon in the control bar or press Space. Supported: MP3, WAV, OGG, FLAC, AAC, M4A.

2 · Pick a preset

Click the preset name to open the browser. Press R for random, ← / → to step through.

3 · Add an image layer

Click Add Layer and drop an image or GIF. It composites over the visualizer in real time.

4 · Preview full screen

Click the expand icon in this topbar (or press \) to hide the panel. Click the canvas to restore.

Image Layers

Up to 5 image or GIF layers, each compositing over the visualizer with its own blend mode and full animation controls.

  • Add Layer — accepts JPG, PNG, GIF, WebP, SVG, AVIF.
  • Drag to reorder — grab the ⠿ handle. Top of list = drawn on top.
  • Solo / Mute — audition one layer or hide it without deleting.
  • Reset ↻ — restores all animation fields to default, keeps the image. Undoable.
  • HD toggle — sets the upload size ceiling: OFF = 1024 px, ON = 2048 px (longest side). Images larger than the ceiling are scaled down proportionally — aspect ratio is always preserved, nothing is cropped or stretched. Images already smaller than the ceiling are passed through untouched (a 300×300 image stays 300×300 regardless of the toggle). Set the toggle before uploading — the resize happens once at upload time and is not reversible without re-uploading. HD layers use ~4× more GPU memory than standard; use HD only when fine detail matters at large projection sizes. GIFs are not resized — they always upload at their native resolution.

Animated GIFs

GIFs animate as real GPU textures — every effect (mirror, tunnel, hue spin, audio reactivity) applies to every frame in real time.

  • Speed slider — 0.25× to 8×. Default is 1.2× because most web GIFs are encoded slow (10 fps native). Drag right for fast-forward, left for slow-motion freeze-frame feel.
  • All effects work — Tile, Mirror, Kaleido, Tunnel, Pulse, Strobe, Hue Spin, Chromatic, Edge — everything applies per frame.
  • GIFs are not resized on upload — they load at native resolution. A 1200×1200 GIF uses significantly more GPU memory than a 512×512 one.
Performance guide:
  • 1–2 GIF layers — smooth on any device.
  • 3–4 GIF layers — fine on a dedicated GPU, may stutter on integrated graphics.
  • 5+ GIFs or large GIFs (1024 px+) — expect slowdown regardless of GPU. Use smaller or fewer-frame GIFs.
  • Adding audio reactivity to a GIF layer costs extra GPU work per frame. If things slow down, reduce the number of audio-reactive GIF layers first.
Tips for smooth GIFs:
  • Use GIFs under 512 px for best performance. Larger GIFs look great but cost more.
  • Fewer frames = faster. A 10-frame loop at 8× looks identical to a 30-frame loop at natural speed and uses 3× less memory.
  • Mute GIF layers you're not currently using — muted layers cost nothing.
  • A dedicated GPU (MacBook Pro, gaming PC) handles more GIF layers automatically — no settings to change.
🚀 GIF Optimizer — Now Available!

When you upload a large GIF (>10 frames, >256px, or >1MB), the optimizer modal appears automatically. Preview frame-by-frame thumbnails with original indices, adjust "Keep every Nth" to reduce frame count, resize to 128/192/256px, and see live GPU memory savings before committing.

Three options: "Apply & Add Layer" uses your optimized settings, "Use As-Is" adds the original GIF unchanged, "Cancel" aborts the upload.

Note: These are general guidelines — every machine is different. Experiment with frame count, size, and the speed slider to find the sweet spot for your GPU. MacBook Airs need more aggressive optimization than MacBook Pros. When in doubt: fewer frames + smaller dimensions = smoother performance.

Transform

  • Size — scales the image or each tile. Pulse adds audio-reactive size.
  • Angle — static rotation offset, composes with Spin.
  • Skew X / Y — shears the image into a parallelogram. Works in all modes.
  • Persp X / Y — projective warp, lines converge to a vanishing point. Persp Y + Tile = infinite floor grid.
  • Width / Height — independent tile cell shape (Tile mode only).

Motion

  • Spin — continuous rotation. Per-tile when Tile ON.
  • Orbit — circular path around anchor. Switch to Lissajous for figure-8 / clover paths.
  • Bounce — bass-driven upward displacement on every beat.
  • Beat Shake — random-direction UV impulse 24×/sec, scales with audio.
  • Sway — sinusoidal horizontal pendulum.
  • Wander — dual-sine organic drift in X and Y.
  • Pan — Drift = continuous scroll; Bounce = ping-pong around anchor.

Tunnel & Depth Stack

  • Tunnel — seamless zoom-through animation. Tile must be ON. Negative = zoom out.
  • Depth — offsets a layer's zoom phase. 0.5 = 180° = maximum parallax between layers.
Classic move: Two layers, same image, same Tunnel speed. Layer 1 Depth = 0, Layer 2 Depth = 0.5. Tiles zoom past each other at different Z depths.

Visual Effects

  • Tint + Hue Spin — colorize and cycle through the full spectrum continuously.
  • Chromatic — RGB channel split with animated offset. Glitchy color fringe.
  • Posterize — Off / 2 / 4 / 8 / 16 color steps. Pop-art quantize effect.
  • Edge / Sobel — 3×3 Sobel kernel replaces image with luminance edge outline. Neon line-art. Best with Tint + Hue Spin.
Best combo: Edge ON · Tint = vivid color · Hue Spin = 0.3 · Blend = Additive.

Mirror

  • Off / H / V / Quad / Kaleido — fold the image symmetrically.
  • Per Tile — folds inside each tile cell.
  • Whole Image — folds the entire tiled field as one unit.

Audio Reactivity

  • Source — Bass / Mid / Treble / Volume. Drives Pulse, Bounce, Shake, Strobe, Opacity Pulse.
  • Curve — Linear / Squared / Cubed / Gate. Shapes how the signal maps to motion.
  • Strobe — hard binary opacity cut. Threshold sets the trigger level.
Tip: One layer on Bass, another on Treble — they react to different parts of the beat for automatic depth.

Workflow

  • Undo / Redo — ⌘Z / ⌘⇧Z. 50-step history. Every action is undoable.
  • A/B compare — hold the A/B button to preview the last saved state.
  • Save — persists preset + images to browser storage. Appears in My Presets.
  • Focus mode — expand icon (or \) hides the panel. Click canvas to restore.

Curated Combos

Neon Line-Art

Edge ON · Tint → vivid color · Hue Spin → 0.3 · Blend → Additive.
Stack a second layer (Edge OFF) underneath at low opacity for a ghost fill behind the outlines.

3D Parallax Tunnel

Two layers, same image, same Tunnel speed.
Layer 1: Depth = 0. Layer 2: Depth = 0.5 + different Tint color.
Tiles zoom past each other at different Z depths — true parallax without any 3D math.

Infinite Floor Grid

Tile ON · Persp Y → 0.6 · Tunnel → 0.5 · Spin → 0.05 · Mirror → Quad.
Add Bounce → 0.15 (Bass) so the floor heaves on every kick.

Pop-Art Pulse

Posterize → 4 · Tint → bright primary · Hue Spin → 0.1 · Bounce → 0.3 · Curve → Cubed.
The Cubed curve means only hard hits trigger the jump — clean on sparse beats, explosive on drops.

Glitch Stack

Layer 1: Chromatic → 0.6 · Beat Shake → 0.08 · Blend → Overlay.
Layer 2: Edge ON · Strobe → 0.8 · Blend → Additive · Source → Treble.
Layer 1 tracks bass, Layer 2 strobes hi-hats — two independent reactive personalities.

Kaleidoscope Vortex

Tile ON · Mirror → Kaleido · Spin → 0.4 · Orbit radius → 0.08 · Hue Spin → 0.2.
Use a symmetrical image (mandala, snowflake) for tightest results. Reduce Spin to 0.1 for a meditative slow spin.

Breathing Logo

Single image, Tile OFF · Size → 0.35 · Pulse → 0.25 · Opacity → 0.9 · Bounce → 0.1 · Source → Bass · Curve → Squared.
Logo swells gently on every beat. Add Hue Spin → 0.05 for a very slow rainbow drift.

Warp Speed Stars

Use a star field or dot texture. Tile ON · Tunnel → 1.8 · Spacing → 0.3 · Size → 0.15 · Blend → Additive.
Crank Tunnel to 2.0 for warp drive on a drop. Pair with a dark MilkDrop preset underneath.

Liquid Mirror

Tile ON · Mirror → Quad, Scope → Whole Image · Sway Amt → 0.08 · Sway Speed → 0.4 · Wander Amt → 0.04.
No Tunnel — the image slowly morphs and sways. Works beautifully with abstract or liquid pour textures.

GIF Strobe Burst

Load an animated GIF · Speed → 0.5 · Strobe → 0.9 · Threshold → 0.4 · Source → Bass · Blend → Additive.
At low GIF speed the frames barely move, then the strobe fires them on every kick for a burst effect.

Diamond Tile Grid

Tile ON · Angle → 45° · Skew X → 0.3 · Skew Y → −0.3 · Width → 0.7 · Height → 0.7 · Spin → 0.03.
Creates a rotating diamond/rhombus tile pattern. Add Mirror → Quad for four-way symmetry.

Depth Hue Shift

Three layers, all the same image, Tunnel = 0.6.
Layer 1: Depth = 0 · Tint = cyan. Layer 2: Depth = 0.33 · Tint = magenta. Layer 3: Depth = 0.66 · Tint = yellow.
Each color plane zooms past the others — full RGB separation in 3D space.

Retro VHS Scan

Layer 1: original image · Blend → Screen · Opacity → 0.6.
Layer 2: same image offset · Chromatic → 0.4 · Posterize → 8 · Pan → Drift · Speed Y → 0.02.
Slow vertical drift + RGB glitch + color banding = VHS tape distortion feel.

Perspective Billboard

Tile OFF · Persp X → 0.4 · Angle → −5° · Size → 0.55 · Bounce → 0.12 · Source → Bass.
Image leans like a roadside billboard that bounces on every beat. Great for logos at live events.

Lissajous Orbit Pair

Two layers, Path → Lissajous. Layer 1: Freq X = 2, Freq Y = 3, Phase = 0. Layer 2: Freq X = 3, Freq Y = 4, Phase = 0.5.
They trace different figure patterns out of sync with each other — hypnotic and never repeating exactly.

Bass Strobe + Treble Shimmer

Layer 1: Opacity Pulse → 0.6 · Source → Bass · Curve → Cubed — hard hits drive a full flash.
Layer 2: Hue Spin → 1.0 · Chromatic → 0.2 · Source → Treble · Blend → Screen — hi-hats drive shimmer.
Set different images and watch them independently react to different parts of the mix.
